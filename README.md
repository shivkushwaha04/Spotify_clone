# Spotify_clone
# Spotify Clone 🎵

A responsive Spotify-inspired web interface built using HTML and CSS.

## Features

* Modern Spotify-like user interface
* Sidebar navigation menu
* Sticky navigation bar
* Music cards and playlists section
* Responsive layout using Bootstrap
* Icons powered by Font Awesome

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* Google Fonts
* Font Awesome CDN

## Project Structure

```text
├── index.html
├── style.css
└── assets/
    ├── logo.png
    ├── backward_icon.png
    ├── forward_icon.png
    ├── card images
    └── player icons
```

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser

## Note

This project is created for learning and practice purposes only. It is a frontend clone inspired by Spotify and is not affiliated with Spotify.
